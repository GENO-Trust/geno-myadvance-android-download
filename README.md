# GENO MyAdvance

Access salary advances, repayments, and support from your Android phone.

## Download

[Download GENO MyAdvance for Android](https://github.com/GENO-Trust/geno-myadvance-android-download/releases/download/v1.0.1/GENO-MyAdvance-v1.0.1.apk)

Tap the download button above on your Android phone. After the download finishes, open the file to install GENO MyAdvance.

If your phone asks for permission to install the app, allow installation from your browser or file manager, then continue.

## What You Can Do

- Register and verify your customer account.
- Check your available salary advance limit.
- Request a salary advance.
- Track advance status, repayment schedules, and payment history.
- Make repayments and check payment status.
- Manage payment methods and deduction mandates.
- Open support tickets, disputes, and complaints.
- Receive customer notifications.

## Need Help?

Contact GENO MyAdvance support if you have trouble installing, signing in, or completing a payment.
